音遊機台卡片註冊教學

2025/03/01

大家好 我是 Gou ～

對於想要遊玩街機音遊的玩家，不管是想要保存遊戲紀錄還是參加活動，卡片是必須的物品。 這一篇文章我會告訴你關於遊戲機台卡片的知識和遊戲帳號的註冊教學。

到底有哪一些卡片？

目前根據不同遊戲公司分為四種卡片

分別是：

遊戲公司 卡片 旗下遊戲
SEGA Aime MAIMAI、CHUNITHM、音擊（ONGEKI）
KONAMI e-amusement pass beatmania IIDX、Sound Voltex、Jubeat
BANDAI NAMCO Bandai Namco Passport 太鼓達人
Taito NESiCA Groove Coastor

看到這裡你可能會想有這麼多卡片，我到底該用哪一張呢？ 事實上，在 2025 你已經不需要擔心這個問題了。 原因就是我們接下來要提到的 「Amusement IC」。

Amusment IC 是什麼？

Amusement IC 是各家遊戲公司聯合在 2018 年推出的機台 IC 卡統一規格。 以往如果想要玩不同遊戲公司的旗下遊戲就得帶好幾張卡片，但有了 Amusement IC 之後，只要帶一張有 Amusement IC 認證的卡就可以了。 因為最初是由（SEGA、KONAMI、BANDAI NAMCO）共同推出的，所以又稱「三社卡」。

可以在哪裡買到呢？

在日本的話，一般來說你都可以在遊戲廳找到販賣遊戲卡片的機器，通常一張為 300 日圓。

而在台灣的話，在問過一些玩家後大致整理一下：

  • 在機廳（湯 X 熊······）通常 150 TWD 一張
  • 網路上購買
  • 請友人送你

在買之前也可以到社群上問一下

卡片資料管理

買到卡片後就可以在機台使用了，但要管理遊戲資料我們還得要先創建帳號。 而且，剛才雖然提到 IC 卡使用了統一規格，但不同遊戲的資料還是保存在不同遊戲公司的伺服器裡（廢話？）。 所以下一步就是根據想玩的遊戲創建帳號。以下會介紹 SEGA、KONAMI、BANDAI NAMCO 的遊戲帳號註冊方法。

每一家遊戲公司的服務雖然不同，但註冊流程基本上是相同的。

具體來說有這幾個步驟：

  1. 首先到機台使用空卡片遊玩一次
  2. 到各遊戲服務器網頁創建遊戲帳號
  3. 將卡片資料登錄到創建好的帳號

Aime（SEGA）

首先是 SEGA 的 Aime， 適用於 MaiMai、CHUNITHM、音擊等機台。

到機台遊玩一次

第一步要做的是用你的新卡片到機台使用一次，通常新卡會有第一次免費遊玩新手教學的優惠。 在機台使用過卡片後，遊戲紀錄就已經保存在卡片中了。下一步就是創建 SEGA ID 來跟服務器連結。

創建 SEGA ID

進到 My Aime 網頁後，點擊左上角的新規登録（註冊帳號）。

另外，我在寫這篇時發現這篇巴哈文章有很清楚的註冊流程，我想只要跟著文章中的步驟註冊 SEGA ID 就可以了。

將卡片資料登錄到 My Aime

用 SEGA ID 登入進 My Aime 後，在右方有你所有卡片的登錄狀況。 要登錄一張新的卡片，點擊右方的登錄。（卡片最多可以登記三張）

接下來，輸入卡片號碼，然後給這張卡片取一個好記的名字。最後按下右下的登錄按鈕就可以了。

將舊卡片資料轉移到新卡

如果哪一天你買了一張新的卡片，你可以選擇把原本的卡片資料轉移到新的卡片，方法也很簡單。 在 My Aime 的管理頁面找到目前使用的卡片，點擊データ移行（轉移資料）。

接下來的步驟也跟上方一樣，輸入新的卡片號碼和名稱就可以順利轉移卡片資料。

e-amusement pass（KONAMI）

接下來是 Konami 的遊戲服務系統 e-amusement。

進到e-amusement網址後，點擊無料登録(註冊帳號)。

註冊 KONAMI ID

進到註冊頁面後，點選下方的註冊 KONAMI ID。

下一步是設定出生年月日和地區，地區推薦設定為日本（Japan），因為很多線上服務需要是日本地區帳號。 接下來填寫基本資料，送出後會收到 Email 的驗證碼回傳。 最後再勾選一些基本條款後，KONAMI ID 就註冊好了。

將卡片資料連結到 KONAMI ID

進入到 e-amusement 首頁後點選右上方的選單欄，點擊 e-amusement pass。

然後點擊勾選的地方 這邊就是登記卡片資料的頁面了，記得要先使用新卡遊玩過遊戲哦。 輸入卡片號碼、自己設定一個 4 位密碼，然後勾選 PASELI 的設定。 這樣卡片資料就登記好了。

PASELI 是 KONAMI 推出的線上儲值服務。儲值 PASELI 點數後，不但可以遊玩機台免投幣直接付款，還有點數回饋。 不過日本以外地區目前無法使用以上功能。

登記好後，實際的遊玩畫面就像這樣。

將舊卡片資料轉移到新卡

卡片資料轉移方式和 SEGA 很類似，在卡片資料保管頁面下方點選引き継ぐ。 接下來同樣輸入新的卡片資料就可以了。
